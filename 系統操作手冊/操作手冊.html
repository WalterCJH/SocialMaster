SocialMaster 系統操作手冊
終端使用者操作指南 · 適用版本 2.0.0

0 安裝runtime

點擊 [windowsdesktop-runtime-10.0.7-win-x64.exe] 安裝

💡
已安裝過可以省略此步驟，如果不確定是否安裝過，可以先嘗試啟動程式，如果缺少runtime會有提示訊息引導安裝。

1 主畫面介紹

程式啟動後顯示主視窗，由上至下分為四個區域：

┌─────────────────────────────────────────────────────────────────┐ │ ▶ 全部啟動 ■ 全部停止 ⏱ 更改選取發文間隔 ＋ 新增帳號 ⟳ 重新整理 │ ├─────────────────────────────────────────────────────────────────┤ │ 平台篩選列 全部 │ Instagram │ X │ Facebook │ ├─────────────────────────────────────────────────────────────────┤ │ 選取 │ 啟動 │ ID │ 名稱 │ 狀態 │ 平台 │ 來源 │ 發文間隔 │ 下次發文 │ 來源設定 │ │─────┼──────┼────┼──────┼──────┼──────┼──────┼──────────┼──────────┼─────────│ │ │ │ 帳號清單（每筆為一個社群帳號） │ │ ├─────────────────────────────────────────────────────────────────┤ │ 執行日誌（即時顯示各帳號的運作記錄） [清除] │ └─────────────────────────────────────────────────────────────────┘

2 首次使用：新增帳號

點擊右上角 ＋ 新增帳號 按鈕，填入以下欄位後按「確定」：

欄位 說明 範例
社群平台 要發文的目標平台（下拉選單） Instagram / X
內容來源 影片抓取的來源平台（下拉選單） Douyin
來源設定 (URL) 抖音創作者的個人頁網址，需包含 sec_user_id 或 /user/ 路徑 https://www.douyin.com/user/MS4w...
自訂名稱 方便辨識的名稱，只有自己看得到，可留空 帳號A-美食
備註 任意備注，可留空 主力帳號
發文間隔 兩篇發文之間的等待時間範圍（分鐘）。 最短必須小於最長。 1150 ～ 5250 分鐘
💡
預設發文間隔為 1150 ～ 5250 分鐘 （約 19 小時 ～ 3.6 天），可依需求調整。建議不要設定過短以避免帳號被平台偵測。

3 啟動與停止帳號

啟動全部帳號

點擊工具列左側 ▶ 全部啟動 ，所有「啟動」欄位打勾的帳號同時開始執行。

停止全部帳號

點擊 ■ 全部停止 ，所有執行中的帳號立即停止。

啟動 / 停止單一帳號

在帳號列上按 右鍵 ，選擇 ▶ 啟動 或 ■ 停止 。

💡
若目前篩選列選的是特定平台（例如「Instagram」）， ▶ 全部啟動 只會啟動該平台的帳號。

4 啟用 / 停用帳號

表格第二欄 「啟動」 是勾選框：

☑ 打勾 帳號處於啟用狀態，按「全部啟動」時會被包含
☐ 取消勾 帳號停用，不參與批次啟動；若原本在執行中，取消勾後會立即停止
⚠️
取消勾後帳號設定仍然保留，下次勾回「啟動」即可繼續執行，不會遺失資料。

5 帳號狀態說明

「狀態」欄位即時顯示每個帳號目前的運作情況：

狀態 說明
停止 Worker 未啟動，帳號閒置中
閒置 剛啟動，正在初始化準備執行
等待登入 瀏覽器已開啟， 需要手動在瀏覽器中登入帳號
下載中 正在從抖音下載影片至本機
上傳中 正在將影片上傳至社群平台（Instagram / X）
模擬人類行為中 上傳完成後，自動瀏覽頁面以模擬正常使用行為
等待中 等待下次發文時間，「下次發文」欄位會顯示預計時間
錯誤 發生問題，請查看底部日誌了解原因

6 首次登入

帳號第一次啟動時，狀態會顯示 等待登入 ，需要手動完成登入：

  1. 程式自動開啟 Chromium 瀏覽器視窗並導向社群平台登入頁
  2. 在瀏覽器中輸入帳號密碼，完成登入（包含雙重驗證等步驟）
  3. 登入成功後，程式自動偵測並繼續執行，狀態切換為 下載中
  4. 之後重啟程式不需再登入，登入 Session 已保存在帳號的 browser/ 目錄中
⚠️
請勿在同一瀏覽器視窗中手動操作或切換帳號，以免影響自動化流程。

7 編輯帳號

有兩種方式開啟編輯視窗：

  • 在帳號列上 雙擊滑鼠左鍵
  • 在帳號列上按右鍵 → 選 ✎ 編輯
ℹ️
編輯模式下， 「社群平台」欄位為唯讀 （顯示為藍色文字，無法更改）。若需要更換平台，請刪除帳號後重新新增。

可修改的欄位：來源設定 URL、自訂名稱、備註、發文間隔。

8 批次修改發文間隔

若要同時調整多個帳號的發文頻率：

  1. 在帳號列最左欄的 「選取」 勾選框打勾（可同時勾選多個帳號）
  2. 工具列的 ⏱ 更改選取發文間隔 按鈕變為可點擊
  3. 點擊後在對話框中輸入新的 最短（分鐘） 和 最長（分鐘）
  4. 點 確定 套用至所有已選取的帳號
⚠️
最短間隔必須小於最長間隔，否則無法確定儲存。

9 平台篩選

畫面頂部篩選列有四個按鈕： 全部 Instagram X Facebook

帳號清單 只顯示對應平台的帳號
全部啟動 / 全部停止 只對目前篩選的平台生效

點選 全部 可回到顯示所有帳號的狀態。

10 右鍵選單

在帳號表格任一列按 右鍵 ，可操作：

選項 說明
▶ 啟動 啟動此帳號的自動化 Worker
■ 停止 停止此帳號的 Worker
✎ 編輯 開啟帳號編輯視窗
📂 開啟帳號資料夾 用檔案總管開啟帳號資料夾（可查看下載的影片、日誌等）

11 執行日誌

畫面底部的日誌面板即時顯示所有帳號的執行記錄，依顏色區分重要程度：

■ 綠色 一般資訊：下載進度、發文成功、等待中等正常狀態
■ 黃色 警告：影片檔案太小、無下載連結、CDN 連結過期等，系統會自動處理
■ 紅色 錯誤：需要注意，可能需要人工介入（例如帳號被登出、網路問題）

點 清除 可清空畫面上的日誌顯示，不影響磁碟上儲存的日誌檔案。

12 最小化與系統托盤

  1. 點擊視窗右上角 最小化 按鈕，程式縮入系統托盤（右下角）， 背景繼續執行
  2. 托盤圖示 雙擊 可還原視窗
  3. 托盤圖示 右鍵 可選「顯示」或「離開」
💡
關閉視窗時，若仍有帳號在執行中，程式會彈出確認視窗，確認後才會完全關閉。

13 重新整理

點擊右上角 ⟳ 重新整理 按鈕，程式會重新掃描 accounts/ 目錄並載入最新帳號設定。

適用情境：手動修改帳號設定檔後，或在程式外新增了帳號資料夾。

14 發文流程說明

帳號啟動後，系統全自動依序執行以下步驟（無需人工介入）：

1
檢查未上傳影片 優先處理已下載但尚未發文的影片（例如上次上傳失敗的內容）
2
從本地快取找影片 讀取已保存的抖音 API 資料，找出尚未發文的影片並下載
3
呼叫 API 取得新影片（快取全部發完才執行） 向抖音 API 請求最新影片清單，儲存後繼續下載
4
上傳至社群平台 自動操作瀏覽器，將影片發布至 Instagram 或 X
5
模擬人類瀏覽行為 上傳後自動捲動瀏覽頁面，降低帳號被偵測為機器人的風險
6
等待下次發文 依設定的發文間隔（最短 ～ 最長分鐘）隨機等待，再重複以上步驟
ℹ️
系統不會重複發布同一支影片。每筆影片發文成功後會記錄在帳號的 content_log.json 中，下次執行時自動跳過已發文內容。
